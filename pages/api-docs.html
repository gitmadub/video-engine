Introduction


FileHost.net provides a simple HTTP API to integrate ours into your business or application.
All requests to the API should be only GET or POST
You can find your API key here - https://filehost.net/settings

Rate limits

API requests are rate limited to 10 per second - for more requests contact us

Response
{"msg":"Too Many Requests","status":"429"}

Local upload

Upload local files using API

GET https://filehost.net/api/upload/server?key={your_api_key}
Parameter
Description
Required
key
Your API Key
Yes
Response
{
  "msg": "OK",
  "server_time": "2026-03-12 04:30:07",
  "status": 200,
  "result": "https://xxx.cloudatacdn.com/upload/01"

Example to use (cURL)
curl -X POST -F 'api_key={your_api_key}' -F 'file=@file.mp4' https://xxx.cloudatacdn.com/upload/01?{your_api_key}
Example to use (HTML)
<form enctype="multipart/form-data" action="https://xxx.cloudatacdn.com/upload/01?{your_api_key}" method="post">
<input type="hidden" name="api_key" value="{your_api_key}">
<input name="file" type="file">
<input type="submit">
</form>
