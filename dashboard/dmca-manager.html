      Uploader DMCA overview

      This manager is only for uploaders. Sending information is optional. If nothing is submitted within 24 hours, the reported file is deleted automatically.

      What uploaders can do

      Open the case, review the complaint details, and send optional information if they want manual review.

      Leave the form blank if there is nothing to add. The system still records that the uploader responded.

      Delete the reported video immediately from the manager if removal is the correct outcome.

      What happens automatically

      When a complaint is received, the video is hidden from public access and the case appears here.

      If the uploader does not respond and does not delete the video, the file is removed after 24 hours.

      Resolved cases remain visible in the history so the uploader can see what happened.

      How this works

      The uploader workflow is intentionally simple and does not require a full legal counter-notice form inside the dashboard.

      Dashboard behavior

      The case appears for the uploader after the file is hidden. Most uploaders will never add extra information, so the form is optional.

      If the uploader does nothing, the file is deleted after the response window ends.

      If the uploader deletes the file themselves, the case closes immediately as uploader deleted.

      Stored information

      Each case keeps a snapshot of the file title, file ID, complaint timestamps, and the case timeline.

      Optional uploader information can include contact email, contact phone, or free-form notes.

      The dashboard shows the complaint source, reference links, and any attached evidence URLs when available.
