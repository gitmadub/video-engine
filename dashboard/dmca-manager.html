      How this works

      The uploader workflow stays simple and uses the same dashboard shell as the rest of the account area. These cards explain the full review flow that used to sit in the overview tab.

      Review flow

      A new complaint opens a case and starts a 24-hour review window.

      During that review window, the reported file stays online unless the uploader deletes it directly.

      Uploader actions

      Open the case to review claimant details, evidence links, and the timeline.

      Send optional contact details or notes, or delete the reported video immediately from the case view.

      Automatic handling

      If no information is submitted and the video is not deleted, the file is removed automatically when the review window expires.

      If optional information is submitted, the case stays open for manual review and remains visible in the manager.

      Dashboard history

      Open cases show every complaint still in progress, and the 24-hour card counts cases that still have an automatic removal timer running.

      Resolved cases remain visible so the uploader can see whether the video was deleted automatically, deleted manually, restored, or closed for another reason.
